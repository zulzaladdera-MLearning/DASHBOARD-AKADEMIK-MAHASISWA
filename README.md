# DASHBOARD-AKADEMIK-MAHASISWA
Dashboard Akademik Mahasiswa di buat oleh Zulza Laddera Aripin Untuk memenuhi kebutuhan tugas mata kuliah data insight di perkuliahan magister informatika UII.

## Sumber Data:
Data yang didapatkan dari private dataset yang sebelumnya sudah dipakai oleh pembuat tugas ini dalam mengerjakan proyek skripsi pada saat masih S1.

## Proses Data
Data yang digunakan untuk membuat visualisasi ini tidak diolah lagi, karena data sudah bersih dan siap digunakan.
